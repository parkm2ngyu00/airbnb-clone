# airbnb-clone

## 사용 기술
- FrontEnd : React, React Query, Chakra UI
- BackEnd : Django, Django Rest Framework
- Infra : Docker, Github Actions

## 브랜치 전략
- Docker hub push 및 배포 : `main`
- 개발 및 테스트 : `develop`

## CI / CD
- Github Acions와 Docker를 사용해 CI/CD 파이프라인 구축
